# edu.contrapaul.com — IB DP Design resources
Clean structured light theme. Education resources for IB DP Design Technology.
## To add a unit page
Create `/units/unit-name.html`. Link to it from the units grid in `index.html`.
## To add an interactive tool
Create `/tools/tool-name.html` as a self-contained HTML activity.
Update the tools list in `index.html` and change the status badge to “Ready”.
## Structure
- `index.html` — main landing page
- `style.css` — all styles
- `units/` — one HTML file per curriculum unit
- `tools/` — self-contained HTML activity files
- `assets/` — diagrams and images
