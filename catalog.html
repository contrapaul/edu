Buttons

Components

Buttons are the primary call-to-action elements. Each variant has distinct hover, focus, and active states.

Variants .btn-{variant}
Sizes .btn-{sm,lg,xl}
Pill Shape .btn-pill
Icon Button .btn-icon
Loading State .btn-loading
ℹ️ Hover & Active Behavior:
• Hover — lifts 1px with a colored shadow matching the button's brand color.
• Active (click) — returns to baseline position with no shadow, giving a "pressed" feel.
• Focus-visible — shows a 3px outline ring for keyboard accessibility.
/* Primary button hover effect */ .btn-primary:hover { background: var(--color-primary-dark); transform: translateY(-1px); box-shadow: 0 4px 12px rgba(79, 70, 229, 0.4); } .btn-primary:active { transform: translateY(0); box-shadow: none; }

Ripple Effect .click-ripple (JavaScript)
// Ripple effect — attach to click handler function createRipple(event) { const btn = event.currentTarget; const ripple = document.createElement('span'); ripple.className = 'ripple'; const rect = btn.getBoundingClientRect(); const size = Math.max(rect.width, rect.height); ripple.style.width = ripple.style.height = size + 'px'; ripple.style.left = (event.clientX - rect.left - size / 2) + 'px'; ripple.style.top = (event.clientY - rect.top - size / 2) + 'px'; btn.appendChild(ripple); ripple.addEventListener('animationend', () => ripple.remove()); }

💡 Easing Reference:
• --ease-out — cubic-bezier(0.16, 1, 0.3, 1) — Fast start, smooth landing. Use for most UI transitions.
• --ease-spring — cubic-bezier(0.34, 1.56, 0.64, 1) — Overshoots and settles. Use for toggles, popups, and playful interactions.
• Durations: Fast = 150ms (micro-interactions), Normal = 250ms (standard transitions), Slow = 400ms (page-level animations).
