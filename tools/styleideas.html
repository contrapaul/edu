STYLE MENU — mix & match options

Most exhibits are a combination of a base, a hover behavior, and accents. Order like a sandwich: pick one from each section (or say "as shown").

Base Surface

  • Glass — translucent blur (#01)
  • Solid dark — flat panel (#03, #12)
  • Light/paper — bright card on dark page (#04, #17)
  • Textured — grid/scanline patterns (#10, #16, #18)
  • Gradient fill — full color (#19)

Border Treatment

  • None / hairline
  • Gradient ring (#02)
  • Neon glow (#03)
  • Thick brutalist (#06)
  • Dashed (#13) or animated orbit (#20)
  • Shaped — clipped corners, folds (#09)

Hover Behavior

  • Lift — rise + shadow (#01, #04)
  • Press — push down on click (#06)
  • Glow up — brighten border/shadow (#03, #16)
  • Shimmer — one light sweep (#15)
  • Spotlight / tilt — cursor-tracking (#07, #08)
  • Reveal — expand extra info (#12)
  • Drift — moving color blob (#11)

Corner Radius

  • Pill-soft — 20-24px (current site look)
  • Medium — 10-14px
  • Sharp — 0-4px (paper, blueprint)
  • Mixed — one corner different (ticket fold)

Accent Color Strategy

  • Brand gradient everywhere (yellow→magenta→orange)
  • One color per tool — each card gets its own hue
  • Category colors — e.g. planners orange, games pink
  • Monochrome + single pop on hover only

Content Extras

  • Emoji icon (as shown) or SVG icon
  • Screenshot thumbnail (#17 style)
  • Category tag pill (#14)
  • Metadata reveal — updated date, etc. (#12)
  • "NEW" ribbon or badge for recent tools

Motion Dial

  • Still — no animation at all
  • Calm — hover-only, short transitions
  • Lively — cursor tracking, sweeps
  • Ambient — subtle always-on animation (orbit border running constantly, slow gradient drift)

Grid & Layout (page-level)

  • Uniform grid — all cards equal size
  • Bento — featured tool spans 2 cells
  • Masonry — varied heights
  • Single featured + row of small
